## 材料清淡
* Arduino uno 官方版 x 1
* 面包板 x 1 
* 人体红外感应电子模块 HC-SR501 
* 色环电阻 10K x1, 220 x3
* 光敏电阻 5528 x1
* 其他材料：公对公 公对母 杜邦线若干，5V电源，磨砂瓶，纸盒子
* 可能用到的工具：热熔胶，双面胶（厚），电工胶，剪刀
